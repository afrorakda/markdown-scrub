# #92 Markdown Scrub

A specialized "text purifier" that washes away Markdown syntax from AI-generated content, leaving only clean, professional plain text.

## Core Philosophy
- **1 Tool / 1 Action**: Focuses entirely on stripping formatting noise to make text ready for emails, messages, and documents.
- **Physical Feedback**: Designed with a "Scrub" ritual to give users a sense of accomplishment and clarity.
- **Privacy First**: No login, no tracking. All text processing happens locally in your browser.

## Key Features
- **Pro-Scrub Engine**: Intelligently removes headers (`#`), bold/italic (`**`, `_`), links (`[text](url)`), code blocks (```), and list markers.
- **Smart Gap Control**: Automatically detects and collapses excessive empty lines (3+ lines) into a single clean gap—perfect for fixing AI-generated spacing.
- **Visual Purity**: Features a dedicated "SCRUB" button that triggers a visual transformation of the text.
- **Mobile First Focus**: Automatically blurs (closes) the keyboard upon scrubbing to let you see the purified result immediately.
- **One-Tap Copy**: Once scrubbed, the action button transforms into a "Copy" button for a seamless workflow.

## Design System
- **Background**: #fff
- **Main Color**: #333
- **Accent**: #2ecc71 (Success Green for the "Cleaned" state)
- **Radius**: 15px (Enforced UI standard)

## How to Use
1. Paste your Markdown-heavy text into the scrubber.
2. Tap the **SCRUB** button to wash away the symbols and collapse extra gaps.
3. Tap **COPY CLEAN TEXT** to save the purified result to your clipboard.

## Privacy Policy
- No data leaves your device.
- No cookies or tracking scripts.
- Browser-only execution.
